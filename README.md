# Steklov ARAP

![](armadillo_cut.gif)

Oras Phongpanangam

Use mix of Laplacian and Steklov (DtN) operators to run ARAP that works for multi-component shapes. 

## Setup

- `uv sync`
- install `steklovnet` separately
- install `torch_mesh_ops` separately
